# 学生管理系统（Python）
基于Python实现的命令行学生信息管理系统，支持增删改查、数据持久化等功能。

## 功能列表
- 学生信息添加（学号、姓名、年龄、专业）
- 学生信息查询（按学号/姓名）
- 学生信息修改/删除
- 数据保存到JSON文件
- 数据从JSON文件加载

## 运行步骤
1. 克隆仓库：`git clone https://github.com/HLW-bit/main.git`
2. 进入项目目录：`cd /d/PBL/node/main`
3. 安装依赖：`pip install -r requirements.txt`
4. 运行程序：`python src/app.py`

## 测试
运行单元测试：`pytest`
